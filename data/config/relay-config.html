配置步骤：
  1. 点击 Chrome 工具栏的拼图图标 🧩
  2. 找到 "OpenClaw Browser Relay"
  3. 点击右侧的 "详细信息"
  4. 向下滚动，找到 "扩展程序选项"
  5. 或直接访问: chrome://extensions/ → 找到插件 → 详细信息 → 扩展程序选项
在选项页面填写：
  • Relay Server URL: ws://127.0.0.1:18792
  • Gateway Token: 点击上方的 Token 框复制
  • 点击 Save 或 保存
